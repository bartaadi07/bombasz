Beállítjuk a kezdősebességet (pixel/frissítés).

III. Háttérképkezelés (13–21. sor)

function updateBackground() {
Deklaráljuk a függvényt, amely a vizuális környezetért felel.
if (level === 1) {
Szigorú egyenlőségvizsgálat: ha az 1. szinten vagyunk.
document.body.style.backgroundImage = "url('src/hatter1.jpg')";
A CSS background-image tulajdonág módosítása az 1. képre.
} else if (level === 2) {
Ha a szint értéke 2.
document.body.style.backgroundImage = "url('src/hatter2.jpg')";
A második szint hátterének beállítása.
} else if (level === 3) {
Ha a szint értéke 3.
document.body.style.backgroundImage = "url('src/hatter3.webp')";
A harmadik szint (webp) hátterének beállítása.
}
Az ágazatok lezárása.
}
A függvény vége.

IV. Szintléptetési logika (22–36. sor)

function updateLevel() {
A nehézségi szintek számításáért felelős logika.
let newLevel = 1;
Lokális segédváltozó inicializálása.
if (score >= 20) { newLevel = 3; }
Ha elértük a 20 pontot, a 3. szintre ugrunk.
else if (score >= 10) { newLevel = 2; }
10 pont felett a 2. szint aktív.
if (newLevel !== level) {
Vizsgálat: történt-e valódi szintváltás?
level = newLevel;
A globális szintváltozó frissítése.
document.querySelector("h1").textContent = `Level ${level}`;
A fejléc frissítése az új szintszámmal.
speed = 3 + (level - 1) * 2;
Sebességképlet: Level 1=3, Level 2=5, Level 3=7.
updateBackground();
A vizuális háttér frissítése.
}
Belső if lezárása.
}
Függvény lezárása.

function spawnUFO() {
Az UFO-kat legyártó függvény.
let ufo = document.createElement("img");
Új <img> elem létrehozása a memóriában.
ufo.src = "src/ufo.gif";
A képforrás (GIF) hozzárendelése.
ufo.style.position = "absolute";
CSS pozicionálás beállítása a szabad mozgatáshoz.

Véletlenszerű függőleges koordináta generálása.
ufo.style.left = "-200px";
Kezdőpont a képernyőn kívül, bal oldalon.

Az UFO hozzáadása a látható HTML dokumentumhoz.

Eltároljuk a tömbben a későbbi mozgatáshoz.

}
Spawn függvény vége.

let ufo = event.target;

VII. Mozgás és Takarítás (69–82. sor)

function moveAll() {
Minden UFO-t mozgató ciklus.
for (let i = ufos.length - 1; i >= 0; i--) {
Biztonságos visszafelé haladó ciklus a tömbmódosításhoz.
let left = parseFloat(ufos[i].style.left);
Aktuális pozíció kinyerése és számmá alakítása.
ufos[i].style.left = left + speed + "px";
Léptetés a sebességgel jobbra.
if (left > window.innerWidth) {
Képernyőelhagyás vizsgálata.
removeUFO(ufos[i]);
Elszökött UFO törlése.
score = Math.max(0, score - 1); updateScore();
Büntetés: Pontlevonás, de nem megy 0 alá.
}
Feltétel vége.
}
Ciklus vége.
}
Függvény vége.

}
